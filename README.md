# TINFO 320 Diagrams

This repository contains diagrams created for T INFO 320 coursework.

## Cloud DaaS Architecture (AWS WorkSpaces)

This diagram shows a Desktop-as-a-Service (DaaS) architecture designed to support four users using Amazon WorkSpaces on AWS.

Components included:
- User devices connecting through the Internet
- Amazon WorkSpaces service
- VPC with a private subnet
- Directory Service for authentication
- Security Groups
- WorkSpaces instances
- Internal company resources (File Server, Database Server, App Server)

Diagram created using Draw.io by James Daniels.
